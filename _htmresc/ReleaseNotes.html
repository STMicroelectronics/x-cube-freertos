X-CUBE-FREERTOS Software Expansion for STM32Cube V1.3.0 / 22-November-2024

Main Changes
  • Release of X-CUBE-FREERTOS V1.3.0 package
    • Add support for STM32WBA, STM32C0, STM32U5, STM32H5, STM32U0, STM32U3 and STM32N6 series
    • Update to FreeRTOS Kernel version 10.6.2
    • Update of Cortex-M CMSIS to version v5.9.0
    • Update to latest versions of stm32u5xx_hal_driver, stm32h5xx_hal_driver, stm32u0xx_hal_driver, stm32wbaxx_hal_driver, stm32c0xx_hal_driver, cmsis_device_u5, cmsis_device_h5, cmsis_device_u0, cmsis_device_wba and cmsis_device_c0.
