X-CUBE-FREERTOS Software Expansion for STM32Cube V1.1.0 / 20-June-2023

Main Changes
  • Release of X-CUBE-FREERTOS V1.1.0 package
    • Add support for STM32WBA and STM32C0 series
    • Update to FreeRTOS Kernel version 10.5.1
    • Update of Cortex-M CMSIS to version v5.9.0
    • Update to latest versions of stm32u5xx_hal_driver, stm32h5xx_hal_driver, cmsis_device_u5 and cmsis_device_h5.
